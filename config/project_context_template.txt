PROJECT CONTEXT TEMPLATE
========================

Use this template to provide project-specific context to the copyeditor.
This information helps the LLM make more relevant and contextually appropriate suggestions.

Copy and fill out the relevant sections below, then pass the text as the
`project_context` parameter to `process_document()`.

---

PROJECT INFORMATION:
-------------------
Project Name: [e.g., "Q4 2024 Healthcare Analytics Report"]
Project Code: [e.g., "HC-2024-Q4-001"]
Client/Organization: [e.g., "HealthSystem Partners"]

DOCUMENT DETAILS:
----------------
Document Type: [e.g., "Annual Report", "Technical Memo", "Policy Brief", "Executive Summary"]
Target Audience: [e.g., "C-suite executives", "Healthcare practitioners", "General public", "Technical staff"]
Primary Purpose: [e.g., "Inform decision-makers about healthcare trends", "Summarize research findings"]

SUBJECT MATTER:
--------------
Primary Topic: [e.g., "Healthcare utilization patterns"]
Secondary Topics: [e.g., "Cost analysis, Patient outcomes, Geographic variations"]
Key Terms to Watch: [e.g., "telehealth, value-based care, social determinants of health"]

SPECIFIC INSTRUCTIONS:
---------------------
Tone: [e.g., "Formal and academic", "Professional but accessible", "Conversational"]
Special Considerations: [e.g., "Document will be published publicly", "Sensitive political content", "Multiple co-authors with different style preferences"]
Priority Concerns: [e.g., "Ensure consistency in terminology", "Check all citations", "Verify data presentation"]

CONTEXT-SPECIFIC STYLE NOTES:
----------------------------
[Add any project-specific style guidance here]
Example:
- This client prefers "impact" as a noun only (not as a verb)
- Use "patient" not "consumer" throughout
- Reference the 2023 Strategic Plan as "the Plan" after first mention


EXAMPLE USAGE:
=============

project_context <- "
PROJECT INFORMATION:
Project Name: Q4 2024 Healthcare Analytics Report
Client: HealthSystem Partners

DOCUMENT DETAILS:
Document Type: Quarterly Report
Target Audience: Hospital executives and board members
Primary Purpose: Present utilization trends and recommend strategic actions

SUBJECT MATTER:
Primary Topic: Healthcare utilization and capacity analysis
Key Terms: telehealth adoption, ED utilization, inpatient admissions, capacity planning

SPECIFIC INSTRUCTIONS:
Tone: Professional and data-driven, but accessible to non-technical executives
Priority Concerns: Ensure all statistics are clearly attributed to sources, verify consistency in terminology across all four quarters

CONTEXT-SPECIFIC STYLE NOTES:
- Client prefers 'emergency department' over 'emergency room'
- Use 'patient visit' not 'encounter' or 'appointment'
- Reference previous quarters as Q1-Q3 2024, not by month ranges
"

results <- process_document(
  file_path = "Q4_report.pdf",
  project_context = project_context,
  system_prompt = system_prompt
)
